# Exchange Rate Market Daily Report


## Requirements

Java 8 or Above


## BUILD PROJECT
```
mvn clean install
```

## Example Run
```
java -jar target/exchange-market-1.0-SNAPSHOT-jar-with-dependencies.jar
```
